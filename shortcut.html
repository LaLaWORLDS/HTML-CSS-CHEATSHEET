A shortcut is a link that points to a program on the computer. Shortcuts allow you to create links to programs in any folder, Start bar, Taskbar, desktop or other locations on the computer.

15 Shortcuts

  1. Add a bookmark:Ctrl + D
  2. Delete:Shift + Delete
  3. Close Browser:Alt + F4
  4. Show Desktop:Show Desktop
  5. Find files or Folder:Window + F
  6. Change Zoom Level:Ctrl + scroll mouse wheel
  7. Reopen Previous Tab in Browser:Ctrl + Shift + T
  8. Switch Between Open Programs:Alt + Tab
  9. Screenshot:Print Screen then paste Ctrl + V into a Word document
  10. Refresh:F5 or Ctrl + R
  11. Go to Previous browser:Alt + backspace
  12. Spelling check:F7
  13. Add "http://www" to your browser's address bar:Ctrl + Enter
  14. Select the adress bar:Ctrl + L, or F6
  15. Launch the window start menu:Windows key or Ctrl + Esc
